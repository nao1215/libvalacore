isAbsolute

Description:

public bool isAbsolute ()

Returns whether this path is absolute.

Example: 

    assert (new Path ("/usr/bin").isAbsolute () == true);
    assert (new Path ("relative/path").isAbsolute () == false);

Returns:

true if the path starts with "/".
