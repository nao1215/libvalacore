addFirst

Description:

public void addFirst (owned T element)

Adds an element to the front of the list.

Example: 

    var list = new LinkedList<string> (GLib.str_equal);
    list.addFirst ("a");
    list.addFirst ("b");
    assert (list.peekFirst () == "b");

Parameters:

element

the element to add.
