Vala.Collections.Deque.peekLast – Valacore Reference Manual

peekLast

Description:

public T peekLast ()

Returns the last element without removing it. Returns null if the deque is empty.

Example: 

    var deque = new Deque<string> (GLib.str_equal);
    deque.addLast ("a");
    deque.addLast ("b");
    assert (deque.peekLast () == "b");
    assert (deque.size () == 2);

Returns:

the last element, or null if empty.
