join

Description:

public static string join (string separator, string[] parts)

Join an array of strings with the specified separator.

Example: 

    assert (Strings.join (", ", {"a", "b", "c"}) == "a, b, c");

Parameters:

separator

the separator string.

parts

the strings to join.

Returns:

joined string.
