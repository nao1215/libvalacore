BufferedWriter

Object Hierarchy:

Description:

public class BufferedWriter : Object

BufferedWriter provides buffered character-output-stream writing. It wraps a GLib.DataOutputStream and offers convenient methods for writing strings and lines, similar to Java's BufferedWriter.

Example: 

    var writer = BufferedWriter.fromFile (new Path ("/tmp/out.txt"));
    writer.writeLine ("Hello");
    writer.writeLine ("World");
    writer.close ();
