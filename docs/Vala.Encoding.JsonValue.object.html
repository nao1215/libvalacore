Description:

public static JsonObjectBuilder object ()

Creates a new JsonObjectBuilder for building JSON objects.

Example: 

    var obj = JsonValue.object ()
        .put ("name", JsonValue.ofString ("Alice"))
        .put ("age", JsonValue.ofInt (30))
        .build ();

Returns:

a new object builder.
