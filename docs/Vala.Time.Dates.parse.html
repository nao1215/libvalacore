public static DateTime? parse (string s, string fmt)

Parses date-time text with the given format.

Example: 

    Vala.Time.DateTime? dt = Dates.parse ("2024-05-10 08:30:45", "%Y-%m-%d %H:%M:%S");
    assert (dt != null);

Parameters:

s

date-time text.

fmt

format text.

Returns:

parsed date-time, or null when parsing fails.
