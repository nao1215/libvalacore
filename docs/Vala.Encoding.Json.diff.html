Description:

public static JsonValue diff (JsonValue a, JsonValue b)

Computes structural differences between two JSON values.

The result is an array of objects: `[{ "path": "$.x", "left": ..., "right": ... }]`

Parameters:

a

left JSON value.

b

right JSON value.

Returns:

array of diff entries.
