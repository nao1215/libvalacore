timeoutMillis

Description:

public HttpRequestBuilder timeoutMillis (int ms)

Sets the request timeout in milliseconds.

If ms <= 0, the timeout floor of 1000 ms is applied. Pass a positive value to use a custom timeout.

Parameters:

ms

timeout in milliseconds.

Returns:

this builder for chaining.
