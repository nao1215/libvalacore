public static int64 size (Path path)

Returns the size of a file in bytes.

Example: 

    var path = new Path ("/tmp/file.txt");
    int64 sz = Files.size (path);

Parameters:

path

path to the file.

Returns:

file size in bytes, or -1 on error.
