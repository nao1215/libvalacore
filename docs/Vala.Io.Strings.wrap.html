wrap

Description:

public static string wrap (string? s, int width)

Wrap the string at the specified width by inserting newlines.

Example: 

    assert (Strings.wrap ("abcdef", 3) == "abc\ndef");

Parameters:

s

the string.

width

the maximum line width.

Returns:

wrapped string.
