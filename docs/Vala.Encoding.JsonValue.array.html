array

Description:

public static JsonArrayBuilder array ()

Creates a new JsonArrayBuilder for building JSON arrays.

Example: 

    var arr = JsonValue.array ()
        .add (JsonValue.ofInt (1))
        .add (JsonValue.ofInt (2))
        .build ();

Returns:

a new array builder.
