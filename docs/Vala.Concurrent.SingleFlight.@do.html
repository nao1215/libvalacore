@do

Description:

public T @do<T> (string key, SingleFlightFunc<T> fn) throws SingleFlightError

Executes the function once for a key and shares the result.

If the same key is already in flight, this call waits for the running execution and returns its result.

Parameters:

key

deduplication key.

fn

function to run.

Returns:

shared result.

Exceptions:

SingleFlightError.INVALID_ARGUMENT

when key is empty.

SingleFlightError.TYPE_MISMATCH

when the key is in flight with another value type.

SingleFlightError.INTERNAL_STATE

when internal entry state is corrupted.
