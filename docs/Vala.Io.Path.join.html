join

Description:

public Path join (string part1, ...)

Joins multiple path components to this path.

Example: 

    var root = new Path ("/home");
    assert (root.join ("user", "docs").toString () == "/home/user/docs");

Parameters:

parts

path components to join.

Returns:

a new Path with all parts joined.
