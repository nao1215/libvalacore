networkDefault

Description:

public static Retry networkDefault ()

Creates recommended retry policy for network operations.

This preset is tuned for transient network failures and rate-limit responses. It uses exponential backoff and jitter to reduce synchronized retries.

Example: 

    Retry retry = Retry.networkDefault ();

Returns:

configured retry policy.
