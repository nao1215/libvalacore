divideWithScale

Description:

public BigDecimal divideWithScale (BigDecimal other, int scale) throws BigDecimalError

Returns quotient with explicit scale.

Quotient is truncated toward zero.

Parameters:

other

divisor.

scale

scale for quotient.

Returns:

quotient.

Exceptions:

BigDecimalError.INVALID_ARGUMENT

when scale is negative.

BigDecimalError.DIVISION_BY_ZERO

when other is zero.

BigDecimalError.SCALE_OVERFLOW

when resulting scale exceeds int range.
