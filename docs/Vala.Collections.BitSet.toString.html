toString

Description:

public string toString ()

Returns a string representation of the BitSet.

The format lists the indices of set bits in braces, e.g. {0, 3, 7}.

Example: 

    var bits = new BitSet (8);
    bits.set (0);
    bits.set (3);
    assert (bits.toString () == "{0, 3}");

Returns:

the string representation.
