Deque

Description:

public Deque (EqualFunc<T>? equal_func = null)

Creates an empty Deque.

The optional equal_func is used by contains() for element comparison. For string elements, pass GLib.str_equal. If null, pointer equality is used.

Example: 

    var deque = new Deque<string> (GLib.str_equal);
    assert (deque.isEmpty ());

Parameters:

equal_func

the equality function for element comparison, or null for pointer equality.
