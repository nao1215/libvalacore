size

Description:

public uint size ()

Returns the number of entries in the map.

Example: 

    var map = new HashMap<string,string> (GLib.str_hash, GLib.str_equal);
    map.put ("a", "1");
    assert (map.size () == 1);

Returns:

the number of key-value pairs.
