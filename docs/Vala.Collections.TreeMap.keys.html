Description:

public ArrayList<K> keys ()

Returns all keys in sorted order as an ArrayList.

Example: 

    var map = new TreeMap<string,string> ((a, b) => {
        return strcmp (a, b);
    });
    map.put ("c", "3");
    map.put ("a", "1");
    map.put ("b", "2");
    var keys = map.keys ();
    assert (keys.get (0) == "a");
    assert (keys.get (1) == "b");
    assert (keys.get (2) == "c");

Returns:

an ArrayList of keys in sorted order.
