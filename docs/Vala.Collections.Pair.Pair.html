Pair

Description:

public Pair (owned A first, owned B second)

Creates a Pair with the given values.

Example: 

    var pair = new Pair<string,string> ("hello", "world");
    assert (pair.first () == "hello");
    assert (pair.second () == "world");

first

the first value.
