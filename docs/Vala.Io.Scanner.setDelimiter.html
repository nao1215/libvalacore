setDelimiter

Description:

public void setDelimiter (string pattern)

Sets the delimiter pattern used to split tokens.

The delimiter is a regular expression pattern. The default delimiter is "\\s+" (one or more whitespace characters).

Example: 

    var scanner = Scanner.fromString ("a,b,c");
    scanner.setDelimiter (",");
    assert (scanner.next () == "a");

Parameters:

pattern

the regex pattern for the delimiter.
