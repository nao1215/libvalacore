split

Description:

public static string[] split (string? s, string delimiter)

Split the string by the specified delimiter.

Example: 

    string[] parts = Strings.split ("a,b,c", ",");
    assert (parts.length == 3);
    assert (parts[0] == "a");

Parameters:

s

the string to split.

delimiter

the delimiter.

Returns:

array of substrings.
