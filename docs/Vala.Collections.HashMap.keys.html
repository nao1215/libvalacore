Description:

public List<unowned K> keys ()

Returns a list of all keys in the map. The order is not guaranteed.

Example: 

    var map = new HashMap<string,string> (GLib.str_hash, GLib.str_equal);
    map.put ("a", "1");
    map.put ("b", "2");
    GLib.List<unowned string> k = map.keys ();
    assert (k.length () == 2);

Returns:

a list of keys.
