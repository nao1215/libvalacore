isDir

Description:

public static bool isDir (Path path)

Returns whether the directory exists in the specified path.

Example: 

    var path = new Path ("/tmp");
    if (Files.isDir (path)) {
        // path is a directory
    }

Parameters:

path

Path to file or directory.

Returns:

true if the path is a directory, false otherwise.
