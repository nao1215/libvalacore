clearBit

Description:

public void clearBit (int index)

Sets the bit at the specified index to 0.

If the index is beyond the current capacity, this is a no-op.

Example: 

    var bits = new BitSet (8);
    bits.set (5);
    bits.clear (5);
    assert (!bits.get (5));

Parameters:

index

the zero-based bit index.
