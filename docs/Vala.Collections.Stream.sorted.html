public Stream<T> sorted (owned ComparatorFunc<T> cmp)

Returns a Stream sorted using the given comparator.

Example: 

    var sorted = stream.sorted ((a, b) => { return strcmp (a, b); });

Parameters:

cmp

the comparator function.

Returns:
