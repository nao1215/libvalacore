trimLeft

Description:

public static string trimLeft (string? s, string cutset)

Remove the specified characters from the left side of the string.

Example: 

    assert (Strings.trimLeft ("xxyhello", "xy") == "hello");

Parameters:

s

the string to trim.

cutset
