Description:

public StringJoiner add (string element)

Adds an element to this joiner.

Example: 

    var joiner = new StringJoiner (", ");
    joiner.add ("hello");
    joiner.add ("world");
    assert (joiner.toString () == "hello, world");

Parameters:

element

the element to add.

Returns:

this StringJoiner for chaining.
