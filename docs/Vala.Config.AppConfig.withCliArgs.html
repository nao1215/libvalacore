withCliArgs

Description:

public AppConfig withCliArgs (string[] args)

Parses CLI args and stores key-value overrides.

Supported forms: - `--key=value` - `--key value` - `--flag` (treated as `true`)

Parameters:

args

CLI argument array.

Returns:

this AppConfig for chaining.
