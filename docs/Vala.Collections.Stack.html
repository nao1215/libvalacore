Description:

public class Stack<T> : Object

A LIFO (Last-In-First-Out) stack backed by a GLib.Queue.

Inspired by Java's Stack and Go's slice-based stack pattern.

Example: 

    var stack = new Stack<string> ();
    stack.push ("a");
    stack.push ("b");
    assert (stack.peek () == "b");
    assert (stack.pop () == "b");
    assert (stack.pop () == "a");
