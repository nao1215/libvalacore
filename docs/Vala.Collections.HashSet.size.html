size

Description:

public uint size ()

Returns the number of elements in the set.

Example: 

    var set = new HashSet<string> (GLib.str_hash, GLib.str_equal);
    set.add ("a");
    set.add ("b");
    assert (set.size () == 2);

Returns:

the number of elements.
