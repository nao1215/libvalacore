must

Description:

public static JsonValue must (JsonValue root, string path) throws JsonError

Returns a value at path or throws when missing.

Example: 

    JsonValue value = Json.must (root, "$.user.id");

Parameters:

root

root value.

path

JSON path expression.

Returns:

value at path.

Exceptions:

JsonError.INVALID_PATH

when path is empty or malformed.

JsonError.NOT_FOUND

when no value exists at path.
