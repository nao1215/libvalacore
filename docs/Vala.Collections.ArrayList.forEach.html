forEach

Description:

public void forEach (owned ConsumerFunc<T> func)

Applies the given function to each element in the list.

Example: 

    var list = new ArrayList<string> (GLib.str_equal);
    list.add ("hello");
    list.forEach ((s) => {
        print ("%s\n", s);
    });

Parameters:

func

the function to apply to each element.
