readAllLines

Description:

public static List<string>? readAllLines (Path path)

Reads the entire contents of a file as a list of lines.

Example: 

    var path = new Path ("/tmp/file.txt");
    var lines = Files.readAllLines (path);

Parameters:

path

path to the file.

Returns:

a list of lines, or null on error.
