clear

Description:

public StringBuilder clear ()

Clears the contents of this builder.

Example: 

    var sb = new StringBuilder.withString ("data");
    sb.clear ();
    assert (sb.length () == 0);
    assert (sb.toString () == "");

Returns:

this StringBuilder for chaining.
