write

Description:

public static string write (ArrayList<ArrayList<string>> data, string separator)

Serializes rows/columns to CSV text.

Fields containing separator, quote, or line breaks are quoted. Quotes inside fields are escaped as double quotes.

Parameters:

data

rows and columns to serialize.

separator

field separator (for example ",").

Returns:

CSV text.
