@set

Description:

public void @set (int index)

Sets the bit at the specified index to 1.

If the index is beyond the current capacity, the BitSet is automatically grown.

Example: 

    var bits = new BitSet (8);
    bits.set (5);
    assert (bits.get (5));

Parameters:

index

the zero-based bit index. Must be >= 0.
