Description:

public void xor (BitSet other)

Performs a bitwise XOR with another BitSet, modifying this BitSet in place.

Example: 

    var a = new BitSet (8);
    a.set (0);
    a.set (1);
    var b = new BitSet (8);
    b.set (1);
    b.set (2);
    a.xor (b);
    assert (a.get (0));
    assert (!a.get (1));
    assert (a.get (2));

Parameters:

other

the other BitSet.
